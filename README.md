# AutoQuiz — Telegram-бот викторина

Это — Telegram-бот викторина по ПДД и автомобилям с системой очков, рейтингом и режимом экзамена.
Здесь можно проверить свои знания ПДД, узнать интересные факты об автомобилях и угадать машину по фото.
Также можно отслеживать свою статистику и прогресс.

1) Тест ПДД — проверь свои знания правил дорожного движения (20 вопросов)
2) Автофакты — узнай интересные факты об автомобилях (15 вопросов)
3) Угадай машину по фото — викторина с фотографиями автомобилей (10 вопросов)
4) Случайная викторина — микс вопросов из разных категорий
5) Режим экзамена — 20 вопросов подряд с итоговой оценкой
6) Профиль — отслеживай свою статистику и прогресс
7) Рейтинг — рейтинг лучших игроков (топ-10)

## СИСТЕМА ОЧКОВ

- ✅ Правильный ответ: +10 очков
- ❌ Неправильный ответ: -5 очков

## КОМАНДЫ БОТА

1) /start — запустить бота и показать главное меню
2) /help — справка по боту
3) /stats — статистика бота
